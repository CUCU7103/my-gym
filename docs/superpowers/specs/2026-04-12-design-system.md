## 1. 디자인 방향

### 톤 & 무드
- **다크 & 모티베이셔널** — 어두운 배경에 블루 액센트, 에너지 있는 느낌
- 20~30대 직장인 타깃에 맞게 과하지 않은 세련된 다크 테마

### 핵심 원칙
- 앱을 열자마자 오늘 기록 상태를 즉시 인식할 수 있어야 한다
- 핵심 액션(기록 버튼)이 시각적으로 가장 강조되어야 한다
- 색상 언어는 파랑 단일 계열로 일관성을 유지한다 (완료 상태도 동일 계열)
- 정보 과부하 없이 필요한 것만 홈에 담는다

## 4. 컴포넌트 스펙

### 4-1. 메인 기록 버튼

**기록 전 상태**
- 배경: `linear-gradient(135deg, #2563EB, #3B82F6)`
- 텍스트: `흰색`, 15px, 700
- 모서리: 20px border-radius
- 그림자: `box-shadow: 0 0 30px rgba(59,130,246,0.4), 0 8px 20px rgba(0,0,0,0.4)`
- 문구: "오늘 운동 기록"

**기록 완료 상태**
- 배경: 동일한 파랑 그라데이션 유지
- 그림자: 글로우 강도 강화 — `box-shadow: 0 0 50px rgba(59,130,246,0.7), 0 8px 20px rgba(0,0,0,0.4)`
- 문구: "✓ 오늘 운동 완료"
- 보조 텍스트: "✦ 오늘도 해냈어요" (버튼 아래, 10px, `#93C5FD`)

> 색상 변경 없이 글로우 강화로 완료 상태를 표현한다. 초록으로 전환하지 않는다.

### 4-2. 오늘 상태 뱃지

**기록 전**
- 배경: `#1a1a1a`, 테두리: `#2a2a2a`
- 텍스트: "아직 기록 안 했어요", 11px, `#888`
- 모서리: 20px (pill)

**기록 완료**
- 배경: `rgba(59,130,246,0.1)`, 테두리: `rgba(59,130,246,0.3)`
- 텍스트: "✓ 오늘 운동 완료!", 11px, `#3B82F6`

### 4-3. 최근 7일 도트

- 도트 크기: 12px × 12px, 원형
- 운동한 날: `#3B82F6`, `box-shadow: 0 0 8px rgba(59,130,246,0.6)`
- 운동 안 한 날: `#222222`, 테두리 `1px solid #333`
- 오늘 (기록 전): 빈 원, `border: 2px solid #3B82F6`, `box-shadow: 0 0 8px rgba(59,130,246,0.4)`
- 오늘 (기록 후): 채워진 파랑, `box-shadow: 0 0 14px rgba(59,130,246,1.0)` (글로우 최대)
- 요일 레이블: 8px, Text Muted / 오늘은 `#3B82F6`

### 4-4. 통계 카드 (이번 주 / 이번 달)

- 배경: `#1a1a1a`, 테두리: `1px solid #2a2a2a`, border-radius: 14px
- 수치: 22px, 700, `#3B82F6`
- 레이블: 9px, Text Muted
- 가로 2열 배치

### 4-5. 주간 목표 프로그레스

- 배경 바: `#2a2a2a`, height 4px, border-radius 2px
- 채움 바: `linear-gradient(90deg, #2563EB, #3B82F6)`
- 달성률 텍스트: "2 / 3회", 10px, `#3B82F6`

### 4-6. 미니 캘린더

- 7열 그리드 (월~일)
- 운동한 날: 배경 `rgba(59,130,246,0.2)`, 텍스트 `#3B82F6`, 하단 도트 4px
- 오늘 날짜: `border: 1px solid #3B82F6`
- 요일 헤더: 7px, Text Muted
- border-radius: 5px per cell

### 4-7. 탭 바

- 탭: 홈 / 기록 / 설정 (3개)
- 배경: `#0d0d0d`, 상단 테두리: `1px solid #1e1e1e`
- 비활성 레이블: Text Muted (`#555`)
- 활성 레이블: `#3B82F6`
- 아이콘: 15~16px 이모지 또는 SVG 아이콘

### 4-8. 설정 행 그룹

- 배경: `#1a1a1a`, 테두리: `1px solid #222`, border-radius: 14px
- 그룹 레이블: 9px, Text Muted, uppercase, letter-spacing 0.8px
- 행 간 구분선: `1px solid #1e1e1e`
- 오른쪽 값: `#3B82F6` / 비활성값: `#444`

### 4-9. 주간 목표 스테퍼

- `−` / `+` 버튼: 22px 원형, 배경 `#2a2a2a`, 텍스트 `#3B82F6`
- 현재 값: 13px, 700, `#3B82F6`

### 4-10. 기록 리스트 아이템

- 배경: `#1a1a1a`, 테두리: `1px solid #2a2a2a`, border-radius: 12px
- 오늘 아이템: 테두리 `rgba(59,130,246,0.4)`, 배경 `rgba(59,130,246,0.05)`
- 왼쪽 도트: 8px, `#3B82F6` (오늘은 글로우 추가)
- 기록 방식 뱃지: "버튼" / "수동" 텍스트, 8px, 둥근 테두리
- 삭제 버튼: "삭제", 9px, Text Muted — 탭 시 확인 모달 표시

## 5. 화면별 레이아웃

### 5-1. 홈 화면

위에서 아래 순서:
1. 오늘 날짜 (center, 11px)
2. 오늘 상태 뱃지 (center, pill)
3. **메인 기록 버튼** (full-width, 핵심)
4. 최근 7일 도트 (7개, space-between)
5. 통계 카드 2열 (이번 주 / 이번 달)
6. 주간 목표 프로그레스 바
7. 이번 달 미니 캘린더
8. 탭 바

### 5-2. 기록 화면

1. 화면 제목 "기록" + 총 기록 수 보조 텍스트
2. 필터 칩 (전체 / 이번 달 / 이번 주)
3. 기록 리스트 (최신순)
4. "+ 과거 날짜 기록 추가" 점선 버튼 (리스트 하단)
5. 탭 바

### 5-3. 설정 화면

1. 화면 제목 "설정"
2. 목표 그룹: 주간 운동 목표 스테퍼
3. 데이터 그룹: 저장 방식 안내 + 총 기록 수
4. 로컬 저장 안내 박스
5. 앱 정보 그룹: 버전
6. 모든 기록 초기화 (danger 버튼)
7. 탭 바

## 6. 인터랙션 & 상태 전환

| 상황 | 변화 |
|------|------|
| 기록 버튼 탭 (첫 기록) | 버튼 글로우 강화, 뱃지 완료로 전환, 오늘 도트 채워짐, 통계 +1 |
| 기록 버튼 탭 (중복) | 버튼 유지, 토스트/스낵바 "오늘은 이미 기록했어요" |
| 주간 목표 달성 | 프로그레스 바 100% 채워짐, 달성 텍스트 변경 |
| 기록 삭제 탭 | 확인 모달 → 삭제 확정 → 리스트 제거 + 통계 재계산 |
| 과거 날짜 추가 | 날짜 피커 → 확정 → 리스트 추가 + 통계 재계산 |

## 7. 미결정 사항 (Open Questions 반영)

PRD의 Open Questions 중 디자인에 영향을 주는 항목:

- **주 시작일**: 캘린더 및 도트 기준. 월요일 시작으로 구현하되 설정 추가 가능
- **수동 추가 진입점**: 기록 화면 하단 버튼으로만 제공 (홈에서는 제외, MVP 단순화)
- **PWA 설치 유도 배너**: 디자인 미포함, 구현 단계에서 별도 처리
